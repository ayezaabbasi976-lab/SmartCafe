# Smart Cafe Ordering System ☕

A C++ Object-Oriented Programming (OOP) based cafe management system developed to manage menu items, customer orders, staff handling, and billing operations.

---

## 📌 Project Description

Smart Cafe Ordering System is a console-based application developed in C++ using OOP concepts.  
The system allows cafe staff to manage menu items, take customer orders, generate bills, and store order data using file handling.

---

## ✨ Features

- Menu Management
- Add Customer Orders
- Billing System with 5% Tax
- Search Menu Items
- Category Filter
- Staff & Customer Classes
- File Handling
- Exception Handling
- Object-Oriented Design
